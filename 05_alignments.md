# Alineamiento de secuencias
> Un alineamiento de secuencias en bioinformática es una forma de representar y comparar dos o más secuencias o cadenas de ADN, ARN, o estructuras primarias proteicas para resaltar sus zonas de similitud, que podrían indicar relaciones funcionales o evolutivas entre los genes o proteínas consultados. Las secuencias alineadas se escriben con las letras (representando aminoácidos o nucleótidos) en filas de una matriz en las que, si es necesario, se insertan espacios para que las zonas con idéntica o similar estructura se alineen ([diapositivas](https://docs.google.com/presentation/d/1AtboTVaLFIHrH_kvl3voFzaJ1KiY3J5F6j0ncJa3Y_A/))
